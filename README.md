## Installation
##### After clone project : 
```$xslt
~ composer update
~ npm install
~ npm install laravel-echo
~ npm install -g laravel-echo-server
~ npm install socket.io-server 
```
##### for config laravel-echo-server
```
~ laravel-eho-server init
```
##### for run project :
```
~ php artisan serve
~ laravel-echo-server start
~ php artisan queue:work
```
